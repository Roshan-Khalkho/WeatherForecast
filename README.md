# WeatherForecast/ Weather App
A simple Weather App built with HTML, CSS, and JavaScript that fetches real-time weather data using the **Open-Meteo API**. This project helped me learn API integration, the Fetch API, Async/Await, JSON handling, and DOM manipulation while building a clean and interactive web application.

---

## Features

* Search weather by city name
* Display current temperature
* Display wind speed
* Show weather code
* Handle invalid city names
* Simple and clean user interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Open-Meteo Geocoding API
* Open-Meteo Weather API

---

## How It Works

1. Enter a city name.
2. Click the Search button.
3. The app finds the latitude and longitude of the city.
4. It requests the latest weather data from the Open-Meteo API.
5. The weather details are displayed on the page.
